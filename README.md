# HANDOFF DOCUMENTATION

## EDITING CONTENT ON ABOUT AND FILM & TV PAGES

Both pages' dynamic content is controlled within content.js, which is located in the root.

### ABOUT

The About page's paragraph content can be edited using the ABOUT_PARAGRAPHS constant. Each comma-separated string is its own paragraph. Bulleted items use the "•" unicode character as a bullet.

### FILM & TV

The Film & TV page's links can be edited using the FILM_AND_TV_LINKS constant. The external links are represented by a list of bracketed pairs -- each pair contains a "label" and a "destination". The labels are what show up as the text content within the link, while the destinations are the external web addresses you want to link to. For bracketed pairs where the destination slot is an empty string (""), the link will automatically direct to the Coming Soon page.
